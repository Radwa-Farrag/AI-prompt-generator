# AI Prompt Generator 

This is a small AI Prompt Generator built with **Node.js** and **Express**, with a simple server-rendered frontend page.

---

## Instructions

To start the project, follow these steps:

1. Open an empty folder where you want to place the project.  
2. Open the terminal and run:  
   git clone <repo_url>
3. Change into the project folder:
   cd <folder_name>
4. Install all dependencies:
   npm install
5. Create a .env file in the root folder and add your OpenAI API key:
   OPENAI_API_KEY=YOUR_API_KEY
6. Start the server:
   npm run dev
7. You should see:
   Server running on http://localhost:3000
8. Open your browser and go to the link above to see the project in action.
